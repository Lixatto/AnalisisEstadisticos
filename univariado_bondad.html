Pruebas de bondad de ajuste

Estas pruebas verifican si la distribución de la variable se ajusta a una distribución teórica específica (por ejemplo, normal, poisson, etc.).

Chi-cuadrado para una muestra

Compara la frecuencia observada en cada categoría con las frecuencias esperadas según una distribución hipotética.

Kolmogorov-Smirnov

Evalúa la máxima diferencia absoluta entre la función de distribución empírica y la teórica.

Shapiro-Wilk

Muy usada para verificar la normalidad de una variable, especialmente con tamaños de muestra pequeños o moderados.

Interpretación básica: si el p-valor es menor a un nivel de significancia (por ejemplo 0.05), se rechaza la hipótesis nula de que “los datos siguen la distribución teórica”.
