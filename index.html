Bienvenido a mi primera web de análisis de datos

El propósito de esta web es compartir con todos aquellos que desean aprender y explorar las definiciones, teoría y ejemplos prácticos sobre estadística y análisis de datos aplicados a las Ciencias Sociales.

Esta web está organizada en tres grandes secciones de análisis estadísticos:

  • Univariados: Se centra en evaluar una sola variable a la vez. Su objetivo principal es describir y resumir las características fundamentales de esa variable, como su distribución, tendencia central, dispersión y forma. También hay análisis inferenciales
  • Bivariados: Examina la relación entre dos variables para determinar si existe una asociación, correlación o dependencia entre ellas. Este tipo de análisis permite explorar cómo una variable puede influir o estar relacionada con otra.
  • Multivariados: Corresponde al estudio simultáneo de tres o más variables para comprender las complejas relaciones y estructuras subyacentes en los datos. Este enfoque permite identificar patrones, reducir la dimensionalidad y construir modelos predictivo.

En cada sección, encontrarás tanto análisis descriptivos como inferenciales, con explicaciones detalladas, ejemplos prácticos y cómo reportar los resultados según las normas APA 7ª edición.
