Bienvenido a mi primera web de análisis de datos

El propósito de esta web es compartir con todos aquellos que desean aprender y explorar las definiciones, teoría y ejemplos prácticos sobre estadística y análisis de datos aplicados a las Ciencias Sociales.

Esta web está organizada en tres grandes secciones de análisis estadísticos:

  • Univariados: Análisis de una sola variable, incluyendo medidas descriptivas como media, mediana, varianza y más.
  • Bivariados: Métodos para analizar relaciones entre dos variables, como correlación y regresión.
  • Multivariados: Técnicas avanzadas para analizar múltiples variables simultáneamente, como el Análisis de Componentes Principales y el clustering.

En cada sección, encontrarás tanto análisis descriptivos como inferenciales, con explicaciones detalladas, ejemplos prácticos y cómo reportar los resultados según las normas APA 7ª edición.
